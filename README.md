# WorkspaceHotKeys

## Basic Git Snippets

This script provide a group of **snippets** for making faster and easier work with **Git** in a **Terminal**.

| Snippet   | Command           |
| --------- | ----------------- |
| gst       | git status        |
| gad       | git add .         |
| gcom      | git commit -m ""  |
| gpush     | git push          |
| gpul      | git pull          |
| gmer      | git merge         |
| gclone    | git clone         |
